Estabelece o contrato para a exibição dos menus de controle de projetos e planos de sprint.

virtual void IAGestaoProjetos::executar ( const Email & email,
const Papel & papel )
pure virtual

Executa o menu principal de gestão de projetos e planos de sprint.

Apresenta as opções de listagem, criação e edição de projetos e planos de sprint. As opções exibidas devem respeitar as restrições de permissões do papel associado ao e-mail.

Parâmetros
emailReferência constante para o e-mail do utilizador autenticado que está operando o sistema.
papelReferência constante para o papel do utilizador autenticado que está operando o sistema.
