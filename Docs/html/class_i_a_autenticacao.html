Interface de apresentação de autenticação (IAA).

Esta classe abstrata estabelece o contrato para a interação com o utilizador durante o fluxo de entrada e login no sistema.

◆ autenticar()

virtual bool IAAutenticacao::autenticar ( Email & email,
Papel & papel )
pure virtual

Executa o menu de autenticação de usuário.

Solicita as credenciais de acesso (e-mail e senha), realiza a validação de formato dos dados inseridos e invoca a camada de serviço para confirmar a autenticidade.

Parâmetros
emailObjeto Email que será preenchido com o identificador do utilizador autenticado em caso de sucesso.
papelObjeto Papel que será preenchido com o papel do utilizador autenticado em caso de sucesso.
Retorna
true se o utilizador for autenticado com sucesso, false caso contrário.
