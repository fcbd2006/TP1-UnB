Descrição detalhada

Representa o domínio de uma data. Esta classe herda da classe base Dominio e garante a integridade de um valor de data, que deve seguir as regras de formatação e validade de datas estabelecidas pelo sistema, inclusive anos bissextos.

Verifica se determinado ano é bissexto.

Parâmetros
anoO ano a ser verificado.
Retorna
Retorna true se o ano for bissexto, ou false caso contrário. Este método atua como uma função auxiliar para o método de validação da classe Data, garantindo que datas como 29 de fevereiro sejam aceitas apenas em anos bissextos.
