Descrição detalhada

Representa o domínio de um Estado (status de progresso). Esta classe herda da classe base Dominio e é responsável por armazenar e garantir a validade do status de uma atividade ou processo no sistema.

Excepções
std::invalid_argumentCaso o estado fornecido seja inválido. Este construtor recebe uma string como argumento e utiliza o método setValor() para armazenar o valor do estado, garantindo que a validação seja realizada automaticamente e que apenas estados válidos sejam aceitos pelo sistema. Lançará uma exceção em caso de falha na validação.
