Detailed Description

Representa o domínio de um Texto.

  • Esta classe herda da classe base Dominio e é responsável por armazenar e garantir a validade de uma string de texto livre (como descrições, comentários, mensagens ou detalhamentos), respeitando as regras de formatação do sistema.
